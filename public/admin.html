Dealership Admin

The server is running successfully. This is a placeholder page for the admin interface.

The following endpoints are available:

Admin Pages
  • /auth - Authentication and Login
  • /cars - Car Management
  • /carousel-images - Carousel Management
  • /upload - File Uploads
API Endpoints
  • /auth/api
    • POST /login - User login
    • POST /forgot-password - Request password reset
    • POST /reset-password - Reset password
  • /cars/api
    • GET / - List all cars
    • POST / - Add new car
    • GET /:id - Get car details
    • PUT /:id - Update car
    • DELETE /:id - Delete car
  • /carousel-images/api
    • GET / - List all images
    • POST / - Add new image
    • DELETE /:id - Delete image
  • /upload/api
    • POST / - Upload files
    • GET /files - List uploaded files
    • DELETE /files/:id - Delete file
